Online Basketball Website

Problem Statement

This website eliminates the need to manage player information and match schedules among coaches and players by organizing data in a coherent manner.

Description

The online website contains information the all the basketball teams in the NBA league. The website lists each team, players of the team, coaches and managers individually. The website holds information about each player, his history, franchises he has been a part of, health, performance in the current season and the previous seasons, diet and training schedules. It also contains information about the coaches, their history and responsibilities. The website is managed by an admin who performs CRUD operations. The fans and spectators are the anonymous users of the website and can read not all information but some information about the players and the staff associated with each team. Coaches have access to all information of each player in his team. Coaches can perform CRUD operations on training schedules, match schedules, match updates and the performance rating of each player in his team in a particular game. The players can read training schedules, match schedules, match updates and the feedback given by the coaches at the end of each game. Coaches can also read information on other players in other teams but cannot edit or delete any information. Players can also read information on other players in other teams but have no rights to perform CRUD operations on other teams. At the end of each season the admin can perform CRUD operations to update each team with the changes made during the auctions and can add additional features which were implemented by the NBA at the start of each season. These changes are update before the start of each season and the changes are available for view to the players and the coaches of each team. These changes are made available to the anonymous users of the website once the team lists are finalized and updated.
